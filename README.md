🚗 Car Showroom Management System

A Java-based console application designed to manage the operations of a car showroom. This project demonstrates the use of Object-Oriented Programming (OOP) concepts such as inheritance, polymorphism, abstraction, and interfaces.

📌 Overview

The Car Showroom Management System allows users to manage:

Showroom details
Car inventory
Customer information

It provides a simple menu-driven interface to perform operations efficiently.

✨ Features
➕ Add new showroom details
🚘 Add and manage car information
👤 Store customer details
📋 View all records
🔍 Search functionality
🧾 Menu-driven console interface
🛠️ Technologies Used
Java
OOP Concepts
Inheritance
Polymorphism
Abstraction
Interfaces
Collections (ArrayList)

▶️ How to Run
1. Clone the Repository
git clone https://github.com/your-username/car-showroom-system.git
2. Navigate to Project Folder
cd car-showroom-system
3. Compile the Program
javac Main.java
4. Run the Program
java Main
